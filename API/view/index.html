# Request Route URL Description
1 Get / API-route home page
2 Post /login Logs in a user
3 Post /register Registers a user
4 Get /users Retrieve all users
5 Put /user/:id Updates a user
6 Delete /user/:id Deletes a user
7 Get /products Fetch all products
8 Get /product/:id Fetch a single product
9 Post /product Add a new product
10 Put /product/:id Update product details
11 Delete /product/:id Delete a product
